# CS5487_PA3

## How to set up environment

First, install PDM (Python Development Module) on your computer. You can find the installation guide [here](https://pypi.org/project/pdm/).

Then, install the required packages by running the following command in the terminal:

```
pdm install
```

## How to run the program

To run the program, run the following command in the terminal:

```
pdm run python main.py
```

or you could just set up your jupyter book and select the pdm's virtual environment to do so.
